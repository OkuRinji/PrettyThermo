# PrettyThermo — Руководство пользователя

## Назначение

PrettyThermo — программа для термодинамического расчёта с использованием комплекса TERMO94. Позволяет задавать параметры расчёта, запускать TERMO94 и просматривать результаты в удобном формате.

## Состав дистрибутива

```
Release/
├── ThermoApp.exe      # Основная программа расчёта
├── CatalogEditor.exe  # Редактор каталога компонентов
└── TERMO/             # Папка с файлами TERMO94
    ├── COMP.PS        # База компонентов
    └── components.json
```

## Установка

1. Скопируйте папку с программой в удобное место (например, `C:\Programs\PrettyThermo\`)
2. Убедитесь, что рядом с `.exe` файлами находится папка `TERMO\`
3. Установите DOSBox (требуется для запуска TERMO94)

### Установка DOSBox

Скачайте и установите DOSBox с официального сайта: https://www.dosbox.com/

После установки укажите путь к DOSBox в настройках программы (по умолчанию: `C:\Program Files (x86)\DOSBox-0.74-3\DOSBox.exe`)

## Запуск программы

### ThermoApp — основной модуль

Дважды кликните на `ThermoApp.exe`

### CatalogEditor — редактор компонентов

Дважды кликните на `CatalogEditor.exe`

## Работа в ThermoApp

### Главное окно

```
┌─────────────────────────────────────────────────┐
│ [📝 Параметры] [📄 Создать .PS] [▶ Запуск] ... │
├─────────────────────────────────────────────────┤
│ Готов к работе | Каталог: 150 компонентов      │
├─────────────────────────────────────────────────┤
│                                                 │
│  Результаты расчёта                            │
│                                                 │
└─────────────────────────────────────────────────┘
```

### Пошаговая инструкция

#### Шаг 1: Загрузка параметров
1. Нажмите кнопку **📝 Параметры**
2. В диалоге укажите:
   - Автора расчёта
   - Директивы (опции расчёта)
   - Параметры PK, PC
   - Компоненты смеси и их концентрации
3. Нажмите **OK**

#### Шаг 2: Создание входного файла
1. В поле имени файла введите имя (например, `New_file`)
2. Нажмите **📄 Создать .PS**
3. Файл `New_file.ps` будет создан в папке `TERMO\`

#### Шаг 3: Запуск расчёта
1. Нажмите **▶ Запуск**
2. Откроется окно DOSBox с запущенным TERMO94
3. Дождитесь завершения расчёта
4. Закройте DOSBox после завершения

#### Шаг 4: Просмотр результатов
1. Нажмите **📊 Результаты**
2. В окне отобразятся:
   - Элементный состав
   - Параметры расчёта (давление, температура и др.)
   - Равновесный состав газовой фазы
   - Конденсированные продукты

#### Шаг 5: Построение графиков (для серий расчётов)
1. Нажмите **📈 Графики**
2. Выберите параметры для отображения:
   - T — Температура (K)
   - I — Энтальпия (кДж/кг)
   - S — Энтропия
   - R — Плотность
   - M — Молярная масса
   - K — Показатель адиабаты
   - V — Объём газовой фазы
   - Z — Доля конденсата
3. Выберите тип графика:
   - Линейный — линия с точками
   - Точечный — отдельные точки
   - Столбчатый — столбчатая диаграмма
4. Нажмите **📊 Построить**
5. Для сохранения нажмите **💾 Сохранить** (PNG, PDF, SVG)

## Работа в CatalogEditor

### Назначение
Редактирование каталога компонентов для TERMO94.

### Главное окно

```
┌─────────────────────────────────────────────────────────────┐
│ [📂 Загрузить JSON] [💾 Сохранить] [➕ Добавить] ...       │
├─────────────────────────────────────────────────────────────┤
│ ID │ Название          │ Формула        │ Энтальпия        │
├────┼───────────────────┼────────────────┼──────────────────┤
│ 1  │ Вода              │ H2O            │ -241.83          │
│ 2  │ Углекислый газ    │ CO2            │ -393.51          │
│ ...│ ...               │ ...            │ ...              │
└─────────────────────────────────────────────────────────────┘
```

### Добавление компонента
1. Нажмите **➕ Добавить**
2. Введите данные:
   - Название (обязательно)
   - Формула
   - Энтальпия образования (кДж/кг)
3. Нажмите **OK**

### Редактирование компонента
1. Выберите компонент в списке
2. Нажмите **✏️ Редактировать** (или двойной клик)
3. Измените данные
4. Нажмите **OK**

### Удаление компонента
1. Выберите компонент
2. Нажмите **🗑️ Удалить**
3. Подтвердите удаление

### Сохранение изменений

**В JSON файл:**
1. Нажмите **💾 Сохранить в JSON**
2. Выберите место сохранения

**В бинарный файл COMP.PS:**
1. Нажмите **💾 Записать в COMP.PS**
2. Выберите файл для записи (обычно `TERMO\COMP.PS`)
3. Компоненты будут записаны с новой нумерацией (1, 2, 3...)

## Форматы файлов

### components.json
Текстовый файл в формате JSON с данными компонентов:
```json
[
  {
    "id": 1,
    "name": "Вода",
    "formula": "H2O",
    "enthalpy": -241.83
  }
]
```

### COMP.PS
Бинарный файл базы данных TERMO94. Редактируется через CatalogEditor.

### .PS файлы расчёта
Входные файлы для TERMO94. Создаются автоматически программой.

### .RES файлы результатов
Результаты расчёта TERMO94. Читаются программой автоматически.

## Частые проблемы

### «Файл каталога не найден»
**Решение:** Убедитесь, что папка `TERMO\` с файлом `components.json` находится рядом с `.exe` файлом.

### «Не удалось запустить расчёт»
**Решение:**
1. Проверьте установку DOSBox
2. Убедитесь, что путь к DOSBox указан верно
3. Проверьте наличие файлов TERMO94 в папке `TERMO\`

### «Файл результатов не найден»
**Решение:**
1. Убедитесь, что расчёт был завершён успешно
2. Проверьте, что файл `.res` создан в папке `TERMO\`
3. Попробуйте запустить расчёт заново

### Программа не запускается
**Решение:**
1. Убедитесь, что все файлы дистрибутива на месте
2. Запустите от имени администратора
3. Проверьте, не блокируется ли программа антивирусом
